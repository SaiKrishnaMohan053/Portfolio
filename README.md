A responsive, single-page portfolio website built with React, React Router, Bootstrap, and Framer Motion animations. Showcase your skills, projects, and contact information in a clean, modern interface optimized for all devices.

## üì• Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/SaiKrishnaMohan053/Portfolio
   cd portfolio
````

2. **Install dependencies**

   ```bash
   npm install
   # or
   yarn install
   ```

3. **Configure EmailJS**

   * Sign up for an EmailJS account.
   * Create an email service and template.
   * Copy your `SERVICE_ID`, `TEMPLATE_ID`, and `USER_ID`.

4. **Start the development server**

   ```bash
   npm start
   # or
   yarn start
   ```

Visit `http://localhost:3000` in your browser to view the site.
